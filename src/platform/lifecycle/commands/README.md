# helios::platform::lifecycle::commands

Commands for platform lifecycle control.

## Commands

| Command | Purpose |
|---------|---------|
| `PlatformInitCommand` | Requests platform initialization |
| `ShutdownCommand` | Requests controlled platform shutdown |

## Notes

These marker commands are usually queued by lifecycle systems and consumed by
runtime managers during commit points.
