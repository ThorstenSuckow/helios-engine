# helios::platform::window::commands

Commands for window lifecycle and buffer presentation.

## Commands

| Command | Purpose |
|---------|---------|
| `WindowCreateCommand<THandle>` | Requests creation/configuration of a window |
| `WindowResizeCommand<THandle>` | Requests or carries resize updates |
| `WindowCloseCommand<THandle>` | Requests controlled window close |
| `SwapBuffersCommand<THandle>` | Requests back/front buffer swap |
