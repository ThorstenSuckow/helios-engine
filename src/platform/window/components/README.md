# helios::platform::window::components

Window-domain ECS components.

## Components

| Component | Purpose |
|-----------|---------|
| `WindowComponent<THandle>` | Stores window runtime/configuration data |
| `WindowCreateRequestComponent<THandle>` | Marks a pending window creation request |
| `WindowShownComponent<THandle>` | Marks that a window has been shown/activated |
