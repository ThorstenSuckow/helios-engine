# helios::engine::rendering::mesh::commands

Commands for mesh upload requests.

This namespace contains lightweight command payloads used to request mesh
uploads, typically in batched form.

## Commands

| Command | Description |
|---------|-------------|
| `MeshBatchUploadCommand<THandle>` | Requests uploading multiple mesh handles in one command |
