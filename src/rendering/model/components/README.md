# helios::model::components

Components for 3D model management and spatial representation.

This module contains components related to 3D models and spatial representation of entities within the game engine.

## Components

| Component | Description |
|-----------|-------------|
| `ModelAabbComponent` | Stores the canonical AABB of a model for collision and culling. |
